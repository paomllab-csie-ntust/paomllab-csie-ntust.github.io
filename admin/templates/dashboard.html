快速指南
如何使用此管理系統
  1. 管理出版物：點擊 "Publications" 新增、編輯或刪除研究論文
  2. 管理成員：點擊 "Members" 管理實驗室成員資訊和畢業狀態
  3. 管理活動：點擊 "Events" 新增或編輯實驗室活動
  4. 上傳圖片：在各個管理頁面中可以上傳成員照片或活動照片
  5. 預覽網站：點擊側邊欄的 "View Website" 查看前端網站效果
提示： 所有修改會即時儲存到 JSON 文件中。記得定期將更新推送到 GitHub 以更新線上網站。
注意： 此管理系統僅供本地使用，請勿部署到公開網路。
Git 操作指令

完成資料更新後，使用以下指令將變更推送到 GitHub：

cd /mnt/e/project/newPao
git add dataset/*.json
git commit -m "Update data: [描述你的更新]"
git push origin main
